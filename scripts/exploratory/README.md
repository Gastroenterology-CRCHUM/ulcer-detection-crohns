# Exploratory scripts

These scripts were used during development but are **not part of the
analysis reported in the paper**. They are included for completeness.

| Script | Purpose |
|--------|---------|
| `train_size.py` | Pipeline C — ulcer size classification (3-class) |
| `prepare_size_manifest.py` | Build manifest for size classifier |
| `hyperparameter_search.py` | Optuna hyperparameter search |
| `run_data_efficiency.py` | Training-set size ablation |
| `eda.py` | Extended EDA figures |
